# OTP Input
- Asked in Myntra, Swiggy, Myntra
- The Number of Input boxes should be Dynamic.
- The Input Box should only accept Number.
- Each Input Box should take Only One Number and If User press another number then it should replace the previous number.
- As soon as Page Loads It should focus on the first OTP Input Box.
- If Input box filled then the next Input box will be focused automatically.
- If hit backspace then the current value must be deleted and it should focus the previous input box.
